# Ateleris .NET Toolkit

This repository contains shared .NET libraries to be used in other .NET projects.

## Usage

Add this repository as a Git submodule to your project repository.

```bash
git submodule add -b {tag_name} https://github.com/ateleris/dotnet-toolkit.git path/to/dotnet-toolkit
```

Then add the shared projects as project reference in your .NET solution.
